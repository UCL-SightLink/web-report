Project Abstract

Revolutionizing Accessibility Through AI-Powered Satellite Analysis

Problem Statement

Accessibility information in satellite imagery remains largely untapped and unstructured, making it challenging for developers and organizations to integrate geographical accessibility data into their applications. Current solutions lack reliable automated methods for detecting and georeferencing critical accessibility features like pedestrian crossings and park benches from aerial imagery.

Our Solution

SightLink provides a comprehensive AI-powered API that processes Digimap-formatted satellite data to identify and georeference accessibility features. Our solution combines YOLO-based object detection with advanced computer vision techniques, implemented through a modern architecture featuring a React-based frontend, Azure-hosted backend, and dedicated database for efficient data management. The system segments images into 20mx20m tiles with 1m overlap, ensuring precise feature detection and location mapping.

Achievement & Impact

Our system achieves high reliability in feature detection while maintaining strict safety standards to prevent false positives. The API has been successfully integrated with accessibility platforms, enabling developers to enhance their applications with accurate geographical accessibility data. This contribution advances the field of automated accessibility mapping and supports the development of more inclusive navigation solutions for wheelchair users and others with mobility needs.
